### 代码

所有人都信任法官，则信任法官的人数为n-1;
法官不信任任何人，为了区每次信任别人的i作为下标ans[i]--;
每次被信任的人本身作为下标自增，判断值和n-1相同时返回下标即可。

```js
var findJudge = function(n, trust) {
    let ans = new Array(n + 1).fill(0)
    for(let i of trust){
        ans[i[0]]--
        ans[i[1]]++
    }
    for(let i = 1; i <= n; i++){
        if(ans[i] == n - 1){
            return i
        }
    }
    return -1
};
```
